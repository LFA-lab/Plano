Sitemark — Convertisseur PDF vers Excel

Build de l'exe : 28/02/2026 16:34

Cette page décrit comment utiliser l’outil Sitemark pour convertir un rapport PDF de pré-commissioning Sitemark en fichier Excel avec les photos intégrées.

1 — Prérequis

  • Télécharger Python 3.11+ : python-3.14.3-amd64.exe
  • IMPORTANT : cocher « Add Python to PATH » lors de l’installation
  • À décocher : « Appliquer la redirection des alias d’exécution d’application vers Python » (évite les conflits avec d’autres logiciels)
  • Vérifier l’installation : ouvrir un terminal, taper python --version

2 — Télécharger l’exe

Télécharger Sitemark.exe : Sitemark.exe.

L’exe ne nécessite pas d’installation : il suffit de le placer où vous voulez (bureau, dossier projet, etc.).

3 — Utilisation

  • Double-cliquer sur Sitemark.exe
  • Cliquer sur « Choisir un PDF Sitemark »
  • Attendre 1 à 2 minutes (téléchargement des photos depuis Sitemark)
  • L’Excel se génère dans le même dossier que le PDF
  • Les photos sont intégrées directement dans l’Excel

4 — Mise à jour (pour Antoine uniquement)

Pour modifier le comportement de l’outil :

  • Modifier sitemark.py dans VS Code
  • Relancer la compilation : pyinstaller --onefile --noconsole --name "Sitemark" sitemark.py
  • Le nouvel exe est dans dist/Sitemark.exe

Vous pouvez aussi utiliser le script BUILD_EXE.bat fourni dans ce dossier pour construire l’exe en une commande.
