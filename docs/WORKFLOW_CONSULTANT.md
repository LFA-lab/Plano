# WORKFLOW_CONSULTANT.md

## Purpose

This guide explains how non‑technical consultants deploy the Microsoft Project Template.

## Consultant Quick Start Guide (Simplified)

### 1️⃣ Download the Template

Go to the GitHub Pages site → download **ModèleImport.mpt**.

### 2️⃣ Enable Macros (Required)

Microsoft Project may show a security warning — this is normal.
The file is provided by the Plano team and is safe.
Click **Enable Content**.

### 3️⃣ Open in Microsoft Project

File → Open → select **ModèleImport.mpt**.

### 4️⃣ Click the Custom Ribbon Button

This prepares the template automatically.

### 5️⃣ Start Working

The template is now ready for use.

## Detailed Workflow

### Step 1 — Download the Latest Template

Microsoft Project may show a macro warning — this is expected.
Enable macros to allow the template to function.

### Step 2 — Open in Microsoft Project

Open the `.mpt` file normally.

### Step 3 — Use the Custom Ribbon Button

This runs the required macros and configures the template.

### Step 4 — Deploy at Client Site

Copy the `.mpt` to the client machine and start using it.

## Common Issues & Warnings

- Use only the official downloaded `.mpt` file.
- Ensure macros are enabled.
- Do not rename or modify internal files.
- Consultants must not edit macros.
